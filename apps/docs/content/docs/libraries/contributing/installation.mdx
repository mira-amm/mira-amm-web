Fork and Clone [Repository](https://github.com/mira-amm/mira-amm-web).


<Tabs groupId="clone-repository" items={['Git', 'GitHub CLI']} persist>
  <Tab value="Git">
```sh title="Shell"
git clone https://mira-amm/mira-amm-web; # clone repo
cd mira-amm-web;                         # Navigate into directory
./install                                # run install script
  ```
  </Tab>
  <Tab value="GitHub CLI">

```sh title="Shell"
gh repo fork mira-amm/mira-amm-web --clone; # fork and clone repo
cd mira-amm-web;                            # Navigate into directory
./install                                   # run install script
```
  </Tab>
  </Tabs>
