## Architecture and Repositories

The MIRA architecture is divided into several core components, each with a specific purpose. Below is an overview of these components and the repositories associated with them:

| Component                                                             | Description                                                                                                                     |
| --------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| [Core Contract](https://github.com/mira-amm/mira-v1-core)             | Implements the foundational AMM logic, handling swaps, liquidity provision, and pool state management.                          |
| [Periphery Scripts](https://github.com/mira-amm/mira-v1-periphery)    | Provides high-level abstractions for interacting with the core contract, including safety mechanisms and swap functions.        |
| [TypeScript SDK](https://github.com/mira-amm/mira-dex-typescript-sdk) | Offers tools for developers to interact with MIRA, including swapping tokens, querying liquidity pools, and managing liquidity. |
| [Rust SDK](https://github.com/mira-amm/mira-v1-rs)                    | Provides a Rust-based toolkit for developers building high-performance applications on top of MIRA.                             |
