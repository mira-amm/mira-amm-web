### Connect Wallet

You can choose between [Fuel](https://fuel.network), [Ethereum](https://ethereum.org/en/), [Solana](https://solana.com) wallets, and more.

![Microchain - Connect Wallet](/connect_wallet.png)

![Microchain - Select Wallet](/select_wallet.png)

### Select tokens

Select a token pair you want to add liquidity for.

These tokens need to have an equal value on your available balance, ex. X $ETH = Y $USDT.

Click on the <kbd> Add liquidity </kbd> button next to the chosen pair.

![Microchain - How to Add Liquidity Step 2](/how_to_add_liquidity_step_2.png)

### Specify amount

Both tokens are adjustable, and will be auto-adjusted.
